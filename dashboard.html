ห้อง ม.5/12

🚀 ขั้นตอนการดีพลอยหน้าเว็บกรอกข้อมูล (Google Web App)

เพื่อให้นักเรียนสามารถกรอกข้อมูลพิกัด GPS อัตโนมัติและอัปโหลดภาพแบบไร้ขีดจำกัดได้อย่างปลอดภัยและสมบูรณ์ ให้ทำตามขั้นตอนการดีพลอยดังนี้:

  1. เปิดไปที่เว็บ Google Apps Script และล็อกอินด้วยบัญชี Google ของครู
  2. สร้างโปรเจกต์ใหม่และนำโค้ดในไฟล์ obec_backend_web_app.js ไปวางทับในไฟล์โค้ด Code.gs
  3. ในหน้าต่าง Google Apps Script ซ้ายมือ ให้กดปุ่ม "+" (ถัดจากคำว่า Files) -> เลือกสร้างไฟล์ "HTML" -> ตั้งชื่อไฟล์ว่า visit_form
  4. นำโค้ดทั้งหมดจากไฟล์ visit_form.html ไปวางในไฟล์ visit_form.html ที่สร้างใหม่
  5. คลิกปุ่ม "Deploy" (มุมบนขวา) -> เลือก "New deployment"
    • Type: เลือกเป็น "Web app"
    • Execute as: เลือกเป็น "Me" (อีเมลของครู)
    • Who has access: เลือกเป็น "Anyone" (ทุกคน)
  6. กด Deploy ยอมรับสิทธิ์การเข้าถึง (Authorize Access) และคัดลอก **Web app URL** ไปแจกให้นักเรียนส่งพิกัดและรูปบ้านได้ทันที!

📜 โค้ดสำหรับหลังบ้าน (obec_backend_web_app.js)

คัดลอกโค้ดสคริปต์นี้ไปวางในไฟล์ Code.gs บน Google Apps Script:

// โค้ดนี้จะไปแสดงใน obec_backend_web_app.js
// กรุณาเปิดไฟล์จริงที่ /home/tor19006/dschool-weekly-report-bot/obec_backend_web_app.js เพื่อคัดลอกไฟล์ตัวเต็ม
